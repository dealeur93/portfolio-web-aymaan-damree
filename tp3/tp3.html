TP3 – Exercices 1 et 3

Exercice 1

Commentaire – Exercice 1

Connaissances mises en jeu

Introduction aux Media Queries (`@media`). J'ai appris à définir des points de rupture (breakpoints) en pixels pour modifier le CSS (couleurs, tailles) selon la largeur du viewport.

Niveau de difficulté estimé

Facile. La logique "Si écran plus petit que X -> Appliquer ça" est très intuitive.

Intérêt de l'exercice

C'est la base absolue du web moderne pour faire des sites compatibles mobile (Responsive Design).

Qualité du résultat

Le changement est instantané quand on redimensionne, c'est satisfaisant de voir la réaction immédiate du site.

Exercice 3

Commentaire – Exercice 3

Connaissances mises en jeu

Création d'un menu interactif sans JavaScript grâce à la pseudo-classe :target. Cela permet de changer le CSS d'un élément quand son ID est présent dans l'URL (ancre).

Niveau de difficulté estimé

Moyenne. C'est une astuce un peu détournée, donc pas le truc le plus logique au début.

Intérêt de l'exercice

Montre qu'on peut gérer des états simples (ouvert/fermé) juste en CSS pour des interfaces légères.

Qualité du résultat

Ça marche nickel, mais ça manque un peu d'accessibilité comparé à un vrai bouton JS contrôlé.
