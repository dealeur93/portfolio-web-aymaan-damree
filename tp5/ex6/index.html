Créer un effet de parallaxe

Cette page montre un véritable effet de parallaxe et explique étape par étape comment le réaliser.

1. Qu’est-ce qu’un effet de parallaxe ?

L'effet de parallaxe est une technique visuelle où l’arrière-plan se déplace plus lentement que le premier plan lors du défilement de la page.

Ce décalage reproduit la manière dont notre œil perçoit la profondeur : les objets proches semblent bouger plus vite que les objets éloignés.

Sur une page web, on obtient cet effet en déplaçant une image selon la position du défilement.

2. Structure HTML

Une section parallaxe contient une image d’arrière-plan et un bloc de texte placé au-dessus.

2. Structure HTML utilisée

Chaque section parallaxe suit cette structure :

<section class="parallax" data-speed="0.3">
  <img src="img/parallax1.jpg" class="bg">
  <div class="overlay"> ... </div>
</section>

L’attribut data-speed permet de modifier la vitesse du mouvement de l’image. Une valeur faible = déplacement lent = effet plus subtil.

3. Le JavaScript

Le script déplace l’image selon le défilement de la page.

3. Fonctionnement du script

Le JavaScript récupère toutes les sections parallaxe, lit leur attribut data-speed et applique un déplacement vertical proportionnel au scroll. 

const sections = document.querySelectorAll(".parallax");

window.addEventListener("scroll", () => {
  sections.forEach(section => {
    const speed = parseFloat(section.dataset.speed);
    const img = section.querySelector(".bg");

    const y = (window.scrollY - section.offsetTop) * speed;

    img.style.transform = `translate(-50%, ${y}px)`;
  });
});

Ainsi, chaque image se déplace à une vitesse différente, ce qui crée un effet visuel dynamique et immersif.
