Bienvenue sur mon site TPS

Aymaan Damree étudiant en BUT2 MMI parcours Création Numérique, je vous présente ici mes travaux de développement Front. Ce site retrace mon évolution à travers différents TPs, où j'ai pu expérimenter, échouer parfois, et surtout apprendre à transformer des maquettes en sites web fonctionnels.

Ce que j'ai appris

Ce semestre a été l'occasion de consolider mes bases et de découvrir des concepts avancés :

  • Flexbox & Grid : Pour des mises en pages complexes et adaptatives.
  • JavaScript interactif : Manipulation du DOM et gestion des événements pour donner vie au site.
  • Rigueur du code : L'importance d'une structure HTML sémantique et d'un CSS maintenable.
  • Animations : Créer du dynamisme avec les transitions CSS et les keyframes.

Les difficultés rencontrées

Tout n'a pas été simple. J'ai dû faire face à plusieurs challenges :

  • Logique JS : Comprendre comment structurer mon code JavaScript pour qu'il soit propre et fonctionnel a demandé du temps.
  • Pixel Perfect : Respecter les maquettes à la lettre demande une attention au détail constante.
  • Débogage : Apprendre à utiliser l'inspecteur pour comprendre pourquoi "ça ne marche pas" a été une compétence clé à développer.
