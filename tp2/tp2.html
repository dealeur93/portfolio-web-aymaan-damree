Commentaire – Exercice 2

Connaissances mises en jeu

Comparaison directe entre CSS Grid et Flexbox. J'ai appris à utiliser grid-template-areas pour définir le squelette global de la page (header, main, footer) de manière visuelle, tout en gardant Flexbox pour aligner les contenus à l'intérieur des zones.

Niveau de difficulté estimé

Moyenne. La syntaxe de Grid est puissante mais plus verbeuse que Flexbox.

Intérêt de l'exercice

Essentiel pour comprendre quel outil utiliser : Grid pour la structure macro, Flexbox pour le micro-alignement.

Qualité du résultat

La structure est super propre et le code CSS est lisible grâce aux "Areas".

Exercice 3

Commentaire – Exercice 3

Connaissances mises en jeu

Positionnement CSS avancé avec position: absolute à l'intérieur d'un référentiel position: relative. Gestion de l'ordre d'affichage (axe Z) avec la propriété z-index pour créer des chevauchements contrôlés.

Niveau de difficulté estimé

Difficile. Comprendre quel bloc sert de référentiel (le parent relatif) demande de une reflexion sur les différents emplacements des éléments.

Intérêt de l'exercice

Indispensable dès qu'on veut sortir du flux standard du document pour faire des effets de design (overlay, badges...).

Qualité du résultat

Visuellement ça marche bien, mais il faut faire gaffe sur mobile que ça ne se chevauche pas n'importe comment.
