Commentaire – Exercice 2

J'ai pu comparer deux méthodes d'intégration : Grid et Flexbox. Si Flexbox est intuitif pour aligner des éléments sur un axe, j'ai trouvé que CSS Grid avec grid-template-areas était incroyablement puissant pour définir une structure globale de page.

Cependant, la syntaxe de Grid reste complexe à retenir pour moi. J'ai eu du mal à comprendre pourquoi certains éléments se déplaçaient tout seuls quand je ne définissais pas explicitement leur place. Au final, je préfère Grid pour le layout général et Flexbox pour le contenu des boîtes.

Commentaire – Exercice 3

Cet exercice était un pur exercice de logique sur le positionnement CSS. Il fallait utiliser position: absolute, relative et jouer avec les `z-index` pour empiler les blocs.

C'est typiquement le genre de tâche où je peux vite m'embrouiller. Comprendre que le "père" doit être en relatif pour que le "fils" absolu se place par rapport à lui n'a pas été immédiat. J'ai dû faire beaucoup d'essais pour que les blocs ne se chevauchent pas n'importe comment. C'est technique, mais utile pour des effets de design spécifiques.
